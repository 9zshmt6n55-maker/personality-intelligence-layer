# Feishu Mature Import

这个目录用于把老代理的飞书通话/妙记材料导入人格驱动内核层。

## 放文件的位置

把从飞书导出的通话文字稿、妙记、会议记录放到：

`imports/feishu/raw/`

优先使用带说话人和时间戳的格式，例如 TXT、SRT、VTT、Markdown、DOCX。

## 导入原则

这些原始对话只用于一次性建模，不作为代理日常运行的长上下文。

导入流程是：

1. 全量读取旧对话。
2. 识别说话人、场景、任务、反馈、成功失败案例。
3. 抽取长期稳定的做事风格、风险边界、沟通方式、主人偏好。
4. 生成成熟人格驱动内核层，跳过初始球状态。
5. 用校准问题验证新代理是否像原来的老代理。

## 不应进入人格层的内容

项目资料、联系人、具体文件名、一次性事实，不直接混入人格层。

这些内容以后应进入可检索资料库；人格层只保留会长期影响行动方式的倾向。
