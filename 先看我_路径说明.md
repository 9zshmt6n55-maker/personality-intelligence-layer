# Quick Path Notes

This repository is portable. Do not rely on the original developer's local paths.

Use the current folder as:

```text
<PIL_ROOT>
```

Before operating, read:

```text
00_AGENT_READ_ME_FIRST.md
```

For old-agent backup creation, read:

```text
PIL_OLD_AGENT_BACKUP_WORKSHEET.md
```

## Safe Entrypoints

Restore a backup into its own profile:

```text
RESTORE_BACKUP_AS_PROFILE.cmd
```

Open all saved profiles:

```text
OPEN_ALL_PROFILES.cmd
```

Open one selected profile:

```text
OPEN_PROFILE.cmd
```

Create a new fresh profile:

```text
NEW_PROFILE_FRESH.cmd
```

Open the root legacy state only for migration/debug:

```text
OPEN_LEGACY_DEFAULT.cmd
```

The root legacy state is not a profile and may already have been overwritten by an imported backup. Prefer `agents/<profile>/`.
